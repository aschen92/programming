Data Structures for Prolog Search Algorithms

Introduction

To complete efficient implementations of various searching algorithms in Prolog you will need to take advantage of some pre-built data structures for implementing queues, sets, and heaps (priority queues). This page gives a quick overview of the necessary libraries. Note, it is not necessary that you understand how these structures are implemented in Prolog; you just need to be able to use them.

Sets

The file sets.pl contains a simple implementation of sets based on the standard assoc library. (Assoc is short for associative list and implements key-value mappings similar to Python dictionaries and Java Maps). To use sets, you must import the module via:

:- use_module(sets).

Operations:

set_empty(?Set)
Create or test for an empty set.
set_add(+Set0, +Element, -Set)
Add Element to Set0 yielding Set.
set_add_new(+Set0, +Element, -Set)
Same as set_add/3, but fails if Element is in Set0.
set_member(-Element, +Set)
Element is a member of Set. Enumerates all elements on backtracking.
set_memberchk(+Element, +Set)
Succeed once if Element is in Set.
set_add_list(+Set0, +List, -Set)
Add items from list to Set0, yielding Set.
list_to_set(+List, -Set)
Create Set from the items in List.
set_to_list(+Set, -List)
Create a list from the elements in Set.

Queues

The file queues.pl implements a FIFO queue. Items enter at the back of the queue and are removed from the front. To use the module, you must import it via:

:- use_module(queues).

Operations:

queue_empty(?Queue)
Create or test for an empty Queue.
dequeue(-Element, +Queue0, -Queue)
Remove front Element from Queue0 leaving Queue.
dequeue_list(-List, +Queue0, -Queue)
Remove a list of items from the front of Queue0 leaving Q1.
enqueue(+Element, +Queue0, -Queue)
Add Element to the back of Queue0 yielding Queue.
enqueue_list(+List, +Queue0, -Queue)
Add the items in List to Queue0 to produce Queue
list_to_queue(+List, -Queue)
Produce Queue containing the items in List.
queue_to_list(+Queue, -List)
Produce the List containing the Elements in Queue.
queue_length(+Queue, -Length)
Get the number of elements in Queue.

Heaps

Heaps are a standard data structure provided by the SWI-prolog library. You can add Key-Value pairs to a heap and efficiently extract the pair having the smallest key. To use the heap module, include an import like this:

:- use_module(library(heaps.pl)).

Operations:

add_to_heap(+OldHeap, +Key, +Datum, -NewHeap)
Inserts the new Key-Datum pair into the heap.
empty_heap(?Heap)
Check or produce an empty Heap.
get_from_heap(+OldHeap, ?Key, ?Datum, -NewHeap)
Returns the Key-Datum pair in OldHeap with the smallest Key, and also a New Heap which is the OldHeap with that pair deleted.
heap_size(+Heap, ?Size)
Reports the number of elements currently in the heap.
heap_to_list(+Heap, -List)
Returns the current set of Key-Datum pairs in the Heap as a List, sorted into ascending order of Keys.
list_to_heap(+List, -Heap)
Takes a list of Key-Datum pairs (such as keysort could be used to sort) and forms them into a heap.
